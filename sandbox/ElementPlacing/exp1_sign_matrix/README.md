## exp1 sign matrix

Minimal single-particle bend diagnostics from the clean pre-chicane state.

Scope:

- one electron
- no radiation
- no space charge
- `PSI = 0`
- `E1 = 0`
- `ANGLE = ±pi/4, ±pi/2`
- both `SBEND` and `RBEND`

Artifacts produced by the runner:

- generated BMAD and OPALX decks
- merged stdout/stderr logs for every run
- `exp1-summary.csv`
- `exp1-summary.md`

Run with:

```bash
/Users/adelmann/git/opalx/.venv-h6/bin/python run_exp1_matrix.py
```
